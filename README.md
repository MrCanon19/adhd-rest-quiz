# Jakiego odpoczynku potrzebujesz?

👉 **[Zrób quiz → mrcanon19.github.io/adhd-rest-quiz](https://mrcanon19.github.io/adhd-rest-quiz/)**

## Co to jest

Krótki quiz, który diagnozuje jakiego odpoczynku Twój układ nerwowy potrzebuje teraz. Nie chodzi o to ile śpisz — chodzi o rodzaj zmęczenia.

**7 typów odpoczynku:**
- 💤 Fizyczny
- 🧩 Poznawczy
- 🔇 Sensoryczny
- 💙 Emocjonalny
- 🫂 Społeczny
- 🌿 Egzystencjalny
- ✨ Kreatywny

Po 14 pytaniach + 1 pytaniu o poczucie winy podczas odpoczynku — dostaniesz top 3 typy z konkretnymi propozycjami co możesz zrobić teraz.
